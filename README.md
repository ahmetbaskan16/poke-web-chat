# Poke Web Chat

A beautifully designed single-page web client for the Poke assistant.

## Features
- Clean, responsive UI built with Tailwind CSS.
- Directly communicates with the Poke V2 API.
- Ready for deployment to Vercel or GitHub Pages.

## Deployment
1. Connect this repository to [Vercel](https://vercel.com).
2. Deploy!
